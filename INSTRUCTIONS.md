# Инструкция по запуску проекта (Vite + React)

Проект был успешно перенесен с Next.js на Vite за его простоту и скорость.

## 1. Установка зависимостей
Откройте терминал в папке проекта и выполните команду:
```bash
npm install
```
(или `pnpm install` / `yarn install`)

Это установит все необходимые пакеты, включая новые (Vite, React Router).

## 2. Запуск в режиме разработки
Для запуска локального сервера разработки выполните:
```bash
npm run dev
```
После этого откройте ссылку, которая появится в терминале (обычно `http://localhost:5173`).

## 3. Сборка проекта (Build)
Чтобы собрать оптимизированную версию для продакшена:
```bash
npm run build
```
Готовые файлы будут находиться в папке `dist`.

## 4. Предпросмотр сборки
Чтобы проверить собранную версию локально:
```bash
npm run preview
```

---

**Заметка:**
Папка `app` больше не используется и может быть удалена после того, как вы убедитесь, что все работает корректно. Весь исходный код теперь находится в папке `src`.
